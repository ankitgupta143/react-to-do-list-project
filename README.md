# To-Do-List Website

# Description

Implemented features to add, edit, and delete tasks in the To-do List.
IImplemented local storage functionality to persist the To-do List data.
Styled the To-do List app using CSS to enhance visual appeal.
Tested the To-do List app to verify proper functionality.


# Technology Stack

HTML5, CSS3 and React.js
